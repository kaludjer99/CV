# CV — Dušan Kaluđerović

Live site: **[kaludjer99.github.io/CV](https://kaludjer99.github.io/CV/)**

---

Online CV built as a single-page HTML application with multi-language support (English / Serbian) and light/dark theme toggle.

## Features

- Responsive layout (Bootstrap 5)
- Language switcher (EN / SR)
- Light / dark theme toggle
- Print-friendly styling

## Tech Stack

- HTML, CSS, JavaScript
- [Bootstrap 5](https://getbootstrap.com/)
- [Bootstrap Icons](https://icons.getbootstrap.com/)
- [Flag Icons CSS](https://flagicons.lipis.dev/)
